# UC-06 User can view saved recordings

## Goal
To allow the user to browse previously saved recordings.

## Preconditions
- The application is running.

## Main Scenario
1. The user opens the saved recordings page.
2. The system displays the saved recordings together with metadata that helps identify the type of each recording.


## Postconditions
- Saved recordings are visible to the user.

## Notes
- An example way of presenting label indicators is shown in the Figma visualization.
- Probably the best way of showing data is in a way sorted by the tim eof starting recording
