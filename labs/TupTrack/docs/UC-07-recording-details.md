# UC-07 User can view recording details

## Goal
To allow the user to view the details of previously saved recordings.

## Preconditions
- The application is running.

## Main Scenario
1. The user opens the saved recordings page.
2. The user selects a recording.
3. The system displays the saved sensor readings and labels for the selected recording.

## Postconditions
- The details of the selected recording are visible to the user.

- An example way of presenting label indicators is shown in the Figma visualization.
