# UC-05 Recording is visualized in real time

## Goal
To allow the user to check whether the recording is being saved correctly.

## Preconditions
- The system is in the recording state.

## Main Scenario
1. The user starts recording.
2. The system displays recording metadata and labels, including:
   - the recording duration
   - visual indicators showing the assigned labels

## Postconditions
- The current recording data is visible to the user.

## Notes
- The recording duration and label indicators should be updated in real time.
- The example way of showing label indiciators is shown in visualsation in figma
