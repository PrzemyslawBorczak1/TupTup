# UC-02 User can start recording sensor readings

## Description
This is the main purpose of the application.

## Goal
To store real-world sensor readings for later analysis.

## Preconditions
- The application is running.
- The system is able to collect sensor data.

## Main Scenario
1. The user starts recording.
2. The system saves new recording with start time
2. The system saves initial Tup State
3. The system starts recording data from the available sensors.


## Postconditions
- Recording Instance is saved and can be accesed by Gui
